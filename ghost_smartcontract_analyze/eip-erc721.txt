10.supportsInterface(ERC165) : 
컨트랙트가 인터페이스를 구현했다면 TRUE를 아니면 FALSE를 반환한다.
input으로는 interfaceId가 들어간다.
eip 페이지에서 각 인터페이스에 상단에 erc-165 indentifier가 있다.
해당 function 가스 소비량은 30,000gas 이하이다.

-확장 인터페이스
1.ERC721TokenReceiver : nft 수신을 다룬다.
erc721은 transfer function 실행 후 onerc721Received function을 실행한다.
해당 function은 transfer을 revert하거나 reject한다.
magic value 이외의 값을 반환하면 트랜잭션은 revert된다.
input으로는 
_operator : safeTransferFrom을 call하는 주소
_from : 이전에 토큰을 소유했던 주소
_tokenid
_data : 추가 데이터

2.ERC721Metadata : nft 세부사항 담음.
function = 
name
symbol
tokenuri

3.ERC721Enumerable : erc721을 열거 확장하는 인터페이스
function =
totalSupply
tokenByIndex
tokenOfOwnerByIndex
